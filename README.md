# Дорога к 70 · PWA

Персональный трекер веса, калорий, шагов и тренировок. Работает офлайн, устанавливается на главный экран iPhone/Android как нативное приложение.

## Что внутри

- `index.html` — приложение целиком (HTML + CSS + JS в одном файле, без зависимостей)
- `manifest.json` — PWA-манифест
- `sw.js` — service worker для офлайн-режима и кэширования
- `icons/` — иконки 192/512/maskable + apple-touch-icon
- `favicon.png` — favicon

## Развёртывание

PWA требует HTTPS. Три простых варианта:

### Вариант 1: Vercel (рекомендую)
```bash
npm install -g vercel
cd fit-tracker-pwa
vercel
```
Готово, получишь HTTPS-ссылку.

### Вариант 2: Netlify Drop
1. Открой https://app.netlify.com/drop
2. Перетащи папку `fit-tracker-pwa` целиком
3. Получишь HTTPS-ссылку

### Вариант 3: GitHub Pages
1. Запушь содержимое в репозиторий
2. В Settings → Pages включи source с `main` branch
3. Открой `https://<username>.github.io/<repo>`

## Установка на iPhone

1. Открой ссылку в Safari (не в Chrome — iOS-PWA работает только из Safari)
2. Нажми кнопку «Поделиться»
3. «На экран Домой»
4. Иконка с твоей кривой снижения веса появится на главном экране

## Установка на Android

В Chrome через 5 секунд после загрузки появится баннер «Установить на главный экран». Или вручную: меню → «Установить приложение».

## Особенности

- **Данные хранятся локально** в localStorage браузера, никуда не отправляются
- **Работает офлайн** — после первой загрузки service worker кэширует все ассеты
- **Тактильная отдача** на устройствах с поддержкой Vibration API
- **Свайп между днями** через стрелки в навигаторе даты, двойной тап на дату — возврат на сегодня
- **Автосохранение** — все изменения пишутся в storage через 350мс после ввода
- **Экспорт** — кнопка внизу выгружает все данные в JSON
- **Прогноз** — после 3+ записей веса показывает дату достижения цели по текущему темпу

## Настройка целей

Открой `index.html`, найди в JavaScript:
```js
const TARGETS = {
  weight: 70,       // целевой вес
  calories: 1800,   // дневная норма калорий
  steps: 9000,      // дневная норма шагов
  startWeight: 85,  // стартовый вес
};
```

## Сброс данных

В консоли браузера: `localStorage.removeItem('k70-data-v1')` и перезагрузка.
